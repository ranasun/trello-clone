# Trello Clone

A trello clone

## Install

```
npm install
```

## Usage

```
npm run dev
npm run build
npm run preview
```

## Technology Stack

This application is built with the following technologies:

-   [Vite](https://vitejs.dev/)
-   [Typescript](https://www.typescriptlang.org/)
-   [React](https://reactjs.org/)
